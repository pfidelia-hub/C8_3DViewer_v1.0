3D Viewer manual

Основные возможности программы "3D Viewer"

Программа предназначена для визуализации .obj файлов, а именно вершин и рёбер.

Для того что бы отобразить obj файл, необходимо указать программе путь до него, для этого на графическом интерфейсе нужно нажать кнопку "Open file", после чего в появившемся окне выбрать указать директорию и сам obj файл. Если все указано верно, то программа отобразит модель. 

Далее можно настроить отображение модели, для этого есть групп кнопок "Move", "Rotation", Background, Line feature.

Управление положением модели на экране осуществляется нажатием групп кнопок Move и Rotate для перемещения и вращения модели соответственно,   измененеие отображения настроивается по осям X, Y, Z. Для изменения скорости передвижения или вращения в поле "speed up" необходимо выставить множитель.

Группа Background позволяет поменять цвет фона объекта, для этого необходимо нажать кнопку "Color back", под кнопкой будет отображаться текущий выбранный цвет фона, после нажатия "Apply" он применится к основному окну отображения.

Группа "Line feature" позволяет менять цвет линий отображаемого объекта, логика идентична Backround.

Когда настроено отображение модели, можно сделать скриншот готовой "отрендереной" модели для этого необходимо нажать кнопку "Screenshot",в открывшемся окне выбрать путь сохранения изображения, в случае невозможности создания скриншота будет выведено сообщение, если скриншот сохдан, то никакого сообщения не будет.
